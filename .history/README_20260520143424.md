# Role-Based Task Manager

A scalable, production-ready task management application with role-based access control, JWT authentication, and Redis caching.

## Features

- 🔐 **Secure Authentication**: JWT-based authentication with refresh tokens
- 👥 **Role-Based Access Control**: Admin and User roles with different permissions
- ✅ **Task Management**: Full CRUD operations with filtering and pagination
- 🚀 **High Performance**: Redis caching for frequently accessed data
- 📱 **Responsive UI**: Modern React frontend with Tailwind CSS
- 🐳 **Docker Ready**: Complete Docker configuration for easy deployment
- 📊 **API Documentation**: Swagger/OpenAPI documentation
- 🛡️ **Security**: Input validation, rate limiting, helmet.js protection

## Tech Stack

### Backend
- Node.js with Express
- PostgreSQL with Sequelize ORM
- Redis for caching
- JWT for authentication
- Swagger for API documentation
- Winston for logging

### Frontend
- React.js
- React Router for navigation
- Axios for API calls
- React Hook Form for form handling
- Tailwind CSS for styling
- React Hot Toast for notifications

## Prerequisites

- Node.js 18+
- PostgreSQL 14+
- Redis 6+
- npm or yarn
